# 다양한 기상 악조건에 강건한 3D Vision 모델 연구

본 저장소는 **경북대학교 컴퓨터학부 종합설계프로젝트2 3팀**의 연구 프로젝트를 위한 Repository입니다.  
본 프로젝트는 **한국전자기술연구원(KETI)**과 협력하여 수행됩니다.

자율주행, 로보틱스, 디지털 트윈 등의 분야에서 **3D Vision 기술(3D Reconstruction, 3D Object Detection)**은 핵심적인 역할을 합니다.  
하지만 대부분의 기존 모델은 **맑은 날씨 데이터셋 중심으로 학습**되어 있어 **눈, 비, 안개, 야간 환경과 같은 기상 악조건에서 성능이 크게 저하되는 문제**가 있습니다.

본 프로젝트에서는 이러한 문제를 해결하기 위해 **악천후 환경에서의 3D Vision 모델 성능을 분석하고 개선하는 연구**를 수행합니다.

# 연구 주제

본 프로젝트에서는 **총 3개의 연구 주제**를 중심으로 연구를 진행합니다.

## 1. Snow 환경에서의 3D Object Detection 케이스 스터디

눈 환경에서 다양한 **3D Object Detection 모델의 성능 변화를 분석**합니다.

- Snow 환경 데이터셋 기반 실험 수행
- 기상 노이즈가 객체 탐지 성능에 미치는 영향 분석
- 모델별 성능 변화 및 취약점 분석

**1저자:** 이상욱

---

## 2. Night / Rain 환경에 최적화된 3D Object Detection 모델 개발

야간 및 우천 환경에서 성능 저하 문제를 해결하기 위한 **3D Object Detection 모델 개선 및 최적화 연구**를 수행합니다.

- Night / Rain 환경 데이터셋 기반 실험
- 기존 3DOD 모델의 성능 저하 원인 분석
- 악천후 환경에 강건한 모델 구조 또는 학습 방법 개발

**1저자:** 최경진

---

## 3. 강설 환경 내 Posed와 Unposed 3DGS의 기상 강건성 및 2D 전처리 적용 효과 비교 연구

최근의 악천후 환경에 대한 3D Gaussian Splatting (3DGS)의 3D 장면 복원 능력을 분석하고 개선하는 연구는 주로 강우 환경에 편중되어 있으며, 카메라 포즈 추정과 형상 복원을 동시에 수행하는 Unposed 3DGS 모델에 대한 분석 또한 부족한 실정입니다. 이에 본 연구는 강설 환경에서의 2D 전처리가 일반 및 Unposed 3DGS 파이프라인의 3D 복원 능력에 미치는 영향을 비교 분석합니다.

- **Snow-Free 데이터셋 구축**: 기존 Free Dataset에 눈(Snow) 레이어를 합성
- **Posed vs Unposed 비교**: 일반 3DGS와 포즈-형상 결합 최적화 모델(LongSplat)의 기상 강건성 측정
- **2D 전처리 효과 검증**: 기상 아티팩트를 사전에 정제하는 전처리 모델이 3D 복원 품질에 미치는 영향 및 시점 불일치 대응력 분석
- **실험 결과**: 2D 전처리 적용 후의 렌더링 품질 회복 효과는 일반 3DGS에서 더 안정적으로 나타났으며, Unposed 모델은 2D 전처리로 인한 오차에 오히려 방해받을 수 있음을 확인 (2026년 한국방송·미디어공학회 하계학술대회 대학생 논문 결과)

**1저자:** 이윤호
