### 3DGS

free dataset을 사용한 LongSplat 모델 결과에 Weather edit으로 날씨 합성 실험을 진행함.

#### 문제1 - 3DGS 자체 퀄리티 저하

카메라 포즈 등의 문제로 LongSplat->3DGS 변환 결과가 Weather edit이 기대하는 입력과 잘 안 맞는 것으로 보임.
3DGS 학습 시 자원을 제한하는 옵션을 사용함.

- 코랩 유료 결제를 통해 자원 제한 없이 재실험 진행 필요

#### 문제2 - 씬 초반/후반부에서 합성 품질이 저하되는 부분이 존재함.

- 품질이 좋은 구간만 선별하여 사용 가능
- Weather Edit 제공 dataset 활용 고려 (직접 합성하는 작업이 불필요한 것으로 보여짐)



#### 주제 구체화

1안 - Posed vs. Unposed 3DGS의 날씨 민감도 및 2D 전처리 의존성 비교
2안(기존) - 비정형 긴 궤적(Casual Long Videos) 환경에서의 악천후 3D 복원 연구
3안 - WeatherGS의 마스킹 기법을 융합한 '렌즈 가림(Lens Occlusion) 보정형' Unposed 3DGS

- 1안을 우선적으로 진행하고 시간적 여유가 있을 시 2안까지 확장할 것
- Unposed + 2D 전처리 결과가 기존 3DGS + 2D 전처리 대비 성능 우위 확보 필요

### 3DOD in snow

지난 실험 결과 보행자 class에서만 악천후 환경에서도 성능이 유지되는 현상이 존재함.
이를 면밀히 분석하기 위해 시각화 자료를 준비해옴.

#### VoxelRcnn vs  mvmm(fusion모델) 비교
Voxel은 퓨전에 비해 라이다에서 멀리 떨어진 물체를 잘 파악하지 못했으며, 퓨전 모델은 노이즈가 잘 발생함.

#### 포인트 유실률 비교
차, 보행자, 자전거 세 가지 class 중 보행자 class만 포인트 유실률이 11.85로 가장 낮음.

- 보행자는 이미 강건하므로 차, 자전거 class의 성능 향상 필요
- 분석 결과 보행자 class만 포인트 유실률이 낮았으므로, 차와 자전거 class에서의 포인트를 복원하는 방향으로 주제 선정
- 전통적인 Interpolation 기법을 적용해 포인트 클라우드를 복원 적용
- 다만 novelty가 있는지 확인해보기 위해 먼저 해당 조건과 유사한 선행 연구가 존재하는지 조사해볼 것
- 악천후 환경이 적용되지 않은 데이터에서의 결과만큼 성능을 개선시키는 것이 목표
