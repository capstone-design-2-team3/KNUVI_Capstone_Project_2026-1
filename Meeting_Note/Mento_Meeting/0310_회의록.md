# 주제 정리

---

## 3DOD

### 주제

- 눈(snow) 환경에 한정하여 3DOD 모델 케이스 스터디  
- 야간(night) 및 우천(rain) 환경에 강건한 3DOD 모델 개발  

- 최초 계획은 멀티모달 기반 3D Object Detection 연구로, LiDAR + Image 데이터를 함께 활용  

- 이미지의 경우 시뮬레이션 코드 적용 시 이미지 별로 눈(snow) 위치가 계속 달라지는 문제가 존재  
  → 유니티 또는 언리얼과 같은 엔진을 사용해 눈 환경을 만들고 시점 별로 캡쳐해서 진행  

- 그러나 시뮬레이션으로 생성한 LiDAR와 유니티/언리얼로 생성한 이미지 간의 매칭 문제로 인해 단일 모달리티(LiDAR 또는 Image) 중심 연구로 방향을 변경할 가능성이 큼  

- 기존 3DOD 데이터셋에는 데이터 용량이 매우 크다는 문제가 존재, 이에 따라 다음과 같은 방안 활용  
  - nuScenes의 mini subset 활용  
  - 시뮬레이션 환경을 이용하여 특정 조건(환경)을 중심으로 데이터 생성  

- LiDAR 베이스라인 코드 존재하나 자원 부족으로 실험 진행을 못하고 있음, 다음주쯤부터 가능할 것으로 예상  

- 비슷한 연구들은 시나리오가 밤(night) 또는 비(rain)에 특정한 경우가 많고 비교적 최근임

## 3DGS

### 주제 후보

- 비(rain) 환경에 따른 3DGS 성능 분석 및 개선 모델 제작  
- 눈(snow)환경에 대한 3DGS 케이스 스터디(+성능 개선)  
- 야간(night)환경에 대한 3DGS 케이스 스터디  

- 날씨 관련 최적화 연구들은 대부분 비(rain)에 한정되어있음  

- 비에 대한 연구는 이미 많기 때문에 성능 최적화가 들어가야 한다 → 성능 개선까지 필요  

- 눈이나 밤의 경우 성능 분석이 먼저 우선  

- 다음주 중으로 최종 주제 선정 예정  

- 해당 주제의 경우 정량평가가 힘들기 때문에 대부분 정성평가로 진행할 예정  
  (ex) 눈, 비 환경에서는 특정 부분 렌더링이 잘 안 되었다  

- 원본 데이터셋 렌더링 결과 vs 기상 환경 시뮬레이션 적용 후 데이터셋을 사용한 렌더링 결과 시각적 비교  

- 3DGS 논문 중 눈, 비를 합성해서 날씨를 변경하는 논문이 존재, 참고할 것

## 공통

- 성능이 진짜로 떨어지는지, 어느정도 떨어지는지를 빠르게 실험해서 해당 연구가 유의미할지에 대해 잘 생각해보기  

- 다음주까지 성능 어느정도로 떨어지는지 관련 논문 조사해보기  

- 실험 기간이 약 한달로 매우 길기 때문에, 매주 실험 계획을 세워서 정리해서 보고하기  

- 다음주 회의는 목요일, 그다음부터는 월요일로 고정
