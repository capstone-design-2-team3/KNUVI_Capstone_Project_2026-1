## 3DOD_Snow - case study

### 조사 내용
- LiDAR 기반 모델 조사
- image 기반 모델 조사

### 실험 결과
- snow augmentation 적용 시 3D bounding box accuracy 감소 확인

### 진행 상황
- PointPillars 평가 완료
- PointRCNN 학습 진행 중
- image 기반 모델은 조사 단계

### 정진우 박사님 피드백
- LiDAR only와 LiDAR + camera 구성 모두 실험 권장

### 최근 연구 트렌드
- multi-sensor
- mono camera
- camera-only 접근 혼재

### 해석 및 가능 연구 방향
- 센서 구성에 따른 성능 저하 차이 분석
- case study 형태 연구 가능
- LiDAR only 성능 감소 분석
- LiDAR + camera 성능 감소 비교

## 3DOD_Night&Rain

- 이후 성능 개선 연구로 확장 가능

## 3DGS

### 현재 진행
- Weather-GS 테스트 수행
- 비/눈 이미지 artifact 제거 성능 확인
- 합성 데이터라 실환경 generalization 불확실

### 최종 연구 주제
- Weather artifact 존재 환경에서 unposed 3D reconstruction 성능 분석 및 개선
- Baseline: LongSplat

### 향후 계획
- image refinement 모델 조사
- LongSplat 실험 수행
- 논문 작성

### 정진우 박사님 피드백
- Input 단계 artifact 제거 방식은 가능하지만
- temporal flickering 및 reconstruction 오류 발생 가능

### 근본적인 방향
- 3D 모델 내부에서 artifact 해결

### 향후 방향
- Input 단계 artifact 제거 방식과
- 3D 모델 내부 artifact 해결 방식을
- 동시에 진행하여 비교 분석
